# Yuxin Xu Personal Homepage

Pure static personal homepage.

## Structure

- `index.html` is the page entry.
- `styles.css` contains the visual system and responsive layout.
- `script.js` handles bilingual copy, scroll reveals, progress, and photography interactions.
- `resources/imgs/optimized/` contains compressed public images used by the site.
- `hidden_resources/` is ignored by Git and keeps private source material out of the published site.

## GitHub Pages

Publish from the repository root. No build step is required.

Before publishing, review the private copy draft in `hidden_resources/content_review.md` and replace the placeholder profile text only after approval.
